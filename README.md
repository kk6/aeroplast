# aeroplast
Transparent PNG conversion (Mainly for Twitter)

## Usage:

### Install packages

```shell
$ pipenv install
```

### Convert image

```shell
$ pipenv run python aeroplast.py convert original.png converted.png
```
### help

```shell
$ pipenv run python aeroplast.py --help
```

## Origin of the name
```
aeroplast == Bubble wrap == 気泡緩衝材 == プチプチ
```
