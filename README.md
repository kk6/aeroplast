# aeroplast
Transparent PNG conversion (Mainly for Twitter)

## Usage:

### Install packages

```shell
$ pipenv install
```

### Invoke commands

```shell
$ pipenv run python -m aeroplast ...
```

or

```shell
$ pipenv run cli ...
```

### Convert image

```shell
$ pipenv run cli convert original.png converted.png
```
### help

```shell
$ pipenv run cli --help
```

## Origin of the name
```
aeroplast == Bubble wrap == 気泡緩衝材 == プチプチ
```
